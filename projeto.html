HTML/CSS/JS - Calculadora

Desenvolvi uma calculadora interativa como parte do meu estudo de desenvolvimento web. Utilizei HTML, CSS e JavaScript para criar uma ferramenta funcional e fácil de usar. A calculadora é capaz de realizar operações básicas de adição, subtração, multiplicação e divisão, além de apresentar recursos adicionais, como botões de apagar e limpar, garantindo uma experiência de usuário intuitiva. A aplicação foi projetada com uma interface limpa e responsiva, permitindo seu uso em diversos dispositivos. Este projeto me proporcionou uma oportunidade valiosa de praticar minhas habilidades de front-end e de criar uma aplicação útil e prática para o dia a dia.
